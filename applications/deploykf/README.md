## How to run
### Ocean
The folder [`../../manifests/core/whirlpool/ocean/deploykf/`](../../manifests/core/whirlpool/ocean/deploykf/) was generated by running:
```bash
docker build -t wave-deploykf .
docker run --rm -it \
    -v ${PWD}/ocean:/tmp/ \
    -v ${PWD}/../../manifests/core/whirlpool/ocean/deploykf/:/usr/src/deploykf/ wave-deploykf \
    deploykf generate \
        --source-version "0.1.3" \
        --values /tmp/values-argocd.yaml \
        --values /tmp/values-deploykf.yaml \
        --values /tmp/values-kubeflow.yaml \
        --output-dir /usr/src/deploykf
docker rmi wave-deploykf
```
